# Marcus Sync

## How It Works

1. Pick a Reminder list and a Markdown file
2. Marcus syncs your tasks automatically on a schedule (or manually)
3. Tasks appear in your chosen format, ready to use

## Requirements

- macOS 13 (Ventura) or later
- Reminders permission (requested on first launch)

## Privacy

Marcus only reads your selected Reminder list and writes to your chosen file. No data leaves your Mac. No accounts, no cloud, no telemetry.
